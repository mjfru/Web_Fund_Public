Lists & Tables

Lists - Unordered & Ordered

Unordered lists, with no number ranking, are created using < ul > < /ul > with list items (< li > < /li > within

An example of an unordered list looks like this:

    Favourite Foods
  • Pizza
  • Sushi
  • Schnitzel

An ordered list uses numbers instead of bullet points and is useful to create a ranking or priority list. The syntax is the same except the open tag will be < ol > < /ol >

An example of an ordered list looks like this:

    To-Do List
  1. Wash the dishes
  2. Vacuum
  3. Take out trash

Tables

Tables are created first by using the < table > element. From there, table rows (< tr > < /tr >) are nested within. After that, table headers (< th > < /th > and table data (< td > < /td > are nested further.

table > table rows > table headers > table data
